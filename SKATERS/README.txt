La cartella di SKATERSS è suddivisa da più sottocartelle e, tranne per quella bootstrap che ci permette di implementare i componenti del framework e per quella immagini in cui sono collocate tutte le immagini utilizzate all'interno delle varie pagine del sito, la struttura di ogni sottocartella è: file PHP, file JavaScript e file CSS.

-In accesso si ha il form per accedere al proprio account attraverso l'inserimento di email e password: nel file PHP è contenuto il codice che permette al sito di collegarsi al database e verificare la presenza dell'account all'interno del database stesso; file JavaScript vi è esclusivamente il collegamento alla pagina registrazione; nel file CSS lo stile della pagina.

-In registrazione si ha il form per registrare il proprio account al sito inserendo le proprie credenziali: nel file PHP è contenunto il codice che permette al database di salvare tutte le credenziali inserite dall'utente creandogli cosi un account; nel file JavaScript sono presenti le funzioni per il controllo e la visualizzazione della password e la funzione che permettono il movimento di alune immagini all'interno della pagina, ; nel file CSS lo stile della pagina.

-In shop si ha lo store del sito, con la visualizzazione dei vari prodotti disponibili e del proprio carrello: nel file PHP è contetnuto il codie che permette di prendere direttamente dal database le varie immagini dei prodotti e di mantenere salvati i prodotti selezionati da parte dell'utente; oltre a shop.php vi sono altre 3 pagine PHP:  <----> ; nel file JavaScript sono presenti le funzioni che permettono di interagire con i prodotti dello shop e con il carrello <------>; nel file CSS lo stile della pagina.

-In videos si ha la sezione video con tutti i video presenti nel sito: nel file PHP  è contetnuto il codie che permette di prendere direttamente dal database i vari video caricati; nel file JavaScript sono presenti le funzioni che permettono di ricercare i video tramite tasitera o di filtrarli tramite varie categorie, e e funzioni per la gestione della pgina a seconda della ricerca effettuata dall'utente.

Infine direttamente nella cartella si ha la pagina index.php dove risiede la home page del sito con tutte le varie anteprime delle altre pagine e in più le varie mappe presenti sul sito. Nel suo file JavaScript vi sono varie funzioni: alcune servono esclusivamente a collegare le varie pagine del sito, altre sono adibite al funzionamento dello scroll sulle mappe e ad alternare quest'ultime, e infine altre gestiscono la sezione "facci una domanda" con l'alert mentre altre gestiscono la visualizzazione o meno dell'immagine utente con il proprio nome in caso di login effettuato.
